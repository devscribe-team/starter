The ACME REST API lets you manage documents and content programmatically. All endpoints return JSON and use standard HTTP methods.

## Base URL

```
https://api.devscribe.com/v1
```

## Authentication

Include your API key in the `Authorization` header on every request:

```bash
Authorization: Bearer YOUR_API_KEY
```

<Callout type="info">
You can generate and manage API keys from your [account settings](/settings/api).
</Callout>

## Response Format

Every successful response wraps data in a consistent envelope:

```json
{
  "data": {},
  "success": true,
  "message": "Request completed successfully"
}
```

Error responses follow the same structure with an error code:

```json
{
  "error": "Resource not found",
  "success": false,
  "code": 404
}
```
